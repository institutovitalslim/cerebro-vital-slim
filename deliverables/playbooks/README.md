## Estrutura padrão
Cada resumo deve seguir esta lógica:
1. Título principal + subtítulo
2. Faixa 1: ponto de entrada / visão geral
3. Faixa 2: dois caminhos ou dois blocos comparativos
4. Faixa 3: implementação prática
5. Faixa 4: blocos finais de síntese
6. Objetivo central

## Regra editorial
- Texto curto, denso e visual
- Frases com alta utilidade prática
- Tom técnico, premium e limpo
- Pensado para virar imagem, PDF ou apresentação depois

# Scientific Research Protocol

## Regra operacional permanente

Sempre que um conteúdo ou carrossel for baseado em pesquisa científica:
1. abrir o paper original sempre que possível
2. extrair dados importantes e defensáveis
3. não inventar números, percentuais ou conclusões
4. registrar um resumo estruturado para consulta futura

## Objetivo

Criar memória científica acumulativa para consulta rápida em contextos clínicos e de conteúdo, com foco principal em:
- emagrecimento
- reposição hormonal
- longevidade
- saúde metabólica
- saúde geral

## Estrutura padrão de registro por paper

- Tema
- Título completo
- Ano
- Tipo de estudo
- População
- Tamanho amostral
- Intervenção/exposição
- Comparador
- Desfechos principais
- Achados quantitativos importantes
- Limitações
- Aplicação clínica prática
- Cautelas para comunicação pública
- Fonte (PubMed/PMC/DOI/URL)

## Regra de uso em pacientes

Quando o assunto de um paciente tocar em evidência científica, consultar primeiro os resumos científicos já registrados antes de responder ou raciocinar.

## Organização sugerida

Salvar resumos em:
- `memory/science/`

Usar nomes curtos e pesquisáveis, por exemplo:
- `creatine-cognition-meta-analysis-2024.md`
- `tirzepatide-obesity-surmount-1.md`
- `estradiol-menopause-cognition-review.md`

## Lições do processo de logo (CASA COM H)

### Problema 1: Contraste
- Fundo preto (#0A0A0A) + dourado fino (stroke) = invisível em telas com brilho baixo
- Solução: usar preenchimento sólido claro (#E8D5A3, #F5F0E8) ou fundo claro com shape escuro

### Problema 2: SVG stroke vs fill
- Logos com `stroke` (linhas) não renderizam bem em conversão PNG/miniaturas
- Logos com `fill` (formas preenchidas) são robustos em qualquer tamanho
- **Regra:** sempre usar formas preenchidas para logos de app

### Problema 3: Texto em app icon
- Texto abaixo do logo some em 120x120 (tamanho real de app icon)
- App icon deve ser só o símbolo (casa + H)
- Full logo (com texto) é para splash screens e materiais grandes

Quando Tiaro diz "busque novas skills" ele quer evolução real, não mais do mesmo. **Instalar 41 skills sem critério não evolui nada.** Depois da limpeza, sobaram 15 skills que realmente entregam valor.

- Skills de design gráfico (logo, SVG, icon, vetorização) não substituem ferramentas profissionais (Illustrator, Figma) ou designer humano
- Image generate (Gemini, DALL-E) não sabe fazer tipografia precisa
- SVG manual sem fonte real é amador
- 100 skills ruins não valem 1 skill que realmente resolve um problema

### Regra de ouro para skills
1. NUNCA instalar sem ler SKILL.md primeiro

4. PREFERIR skills locais/seguras vs APIs externas
5. TESTAR antes de assumir que funciona

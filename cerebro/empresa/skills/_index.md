# Skills da Empresa (cross-area)

Skills que não pertencem a uma área específica — servem a empresa como um todo.

| Skill | O que faz |
|-------|-----------|
| `relatorio-rotinas` | Lista status de todas as rotinas automáticas (crons) |
| `twitter-banner-creator` | Gera banners de Meta Ads no formato screenshot de tweet/post do X |

Para skills específicas de cada área, consulte `areas/[área]/skills/`.

# Skills da Empresa — leitura estrutural

## Leitura recomendada
1. `_index.md`
2. `GOVERNANCA-SKILLS.md`
3. skill específica (`SKILL.md`)
4. scripts/references/assets conforme necessidade

## Regra estrutural
- `SKILL.md` = interface principal da skill
- `scripts/` = execução auxiliar
- `references/` = apoio e documentação complementar
- `assets/` = artefatos e insumos técnicos/visuais

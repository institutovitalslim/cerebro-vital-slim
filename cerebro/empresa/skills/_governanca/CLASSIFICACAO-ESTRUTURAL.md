# Classificação Estrutural — Skills

## Camadas
### 1. Skill principal
Arquivo `SKILL.md` que define propósito, uso, dependências, critérios e output.

### 2. Scripts auxiliares
Código usado para executar partes da skill. Não substitui a skill principal como interface de leitura.

### 3. Referências
Guias, anatomy, prompts, brand rules e materiais de apoio.

### 4. Assets
Imagens, schemas, exemplos e artefatos visuais/técnicos.

## Regra
Ao navegar uma skill, a ordem de leitura deve ser:
1. `SKILL.md`
2. dependências canônicas citadas na skill
3. scripts/references necessários
4. assets/examples se fizer sentido

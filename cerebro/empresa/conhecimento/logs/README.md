# Logs de conhecimento — camada histórico / log

Este diretório guarda evidências, lições registradas e histórico de execução.

## Uso correto
- consultar para rastreio histórico
- revisar aprendizados passados
- recuperar contexto de evolução

## Não usar como fonte primária
Quando houver conflito, logs não substituem:
- contexto canônico
- playbook operacional
- skill governada

===== higgsfield generate --help
Create jobs, estimate cost, inspect results, and list recent image, video, or text generations.

Usage:
  higgsfield generate [command]

Aliases:
  generate, gen

Examples:
  higgsfield generate create nano_banana_2 --prompt "studio product photo"
  higgsfield generate cost nano_banana_2 --prompt "studio product photo"
  higgsfield generate wait <job_id>

Available Commands:
  cost        Estimate generation cost
  create      Create a generation job
  get         Show one generation job
  list        List recent generation jobs
  wait        Poll a job until it finishes

Flags:
  -h, --help   help for generate

Global Flags:
      --json       print raw JSON responses
      --no-color   disable color output

Use "higgsfield generate [command] --help" for more information about a command.
===== higgsfield generate create --help
Create an image, video, or text job.
Pass params as --name value. Media flags: --image, --start-image, --end-image, --video, --audio.
Media flags accept a UUID (upload id or job id) or a local file path — paths are auto-uploaded.
Use --wait to block until each job finishes and print the result URL(s).

Usage:
  higgsfield generate create <job_set_type> [--param value]... [flags]

Examples:
  higgsfield generate create nano_banana_2 --prompt "cinematic product photo" --image ./photo.png
  higgsfield generate create nano_banana_2 --prompt "..." --wait
  higgsfield generate create <video_model> --prompt "..." --wait --wait-timeout 20m --wait-interval 5s

Flags:
  -h, --help   help for create

Global Flags:
      --json       print raw JSON responses
      --no-color   disable color output

RETURN=0

  higgsfield auth [command]

  higgsfield auth login
  higgsfield auth token
  higgsfield auth logout

Available Commands:
  login       Start browser-based device login
  logout      Delete the locally stored access token
  token       Print the current access token

Flags:
  -h, --help   help for auth

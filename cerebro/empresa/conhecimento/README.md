# Conhecimento — ponte entre canônico, operacional e histórico

Este diretório concentra materiais de conhecimento da empresa.

## Subcamadas práticas
- conteúdo ativo e reaproveitável → operacional
- índices técnicos e logs → histórico/técnico
- material promovido a regra → deve subir para camada canônica adequada

## Regra
Conhecimento útil não é automaticamente canônico. Só vira fonte principal quando for promovido explicitamente.

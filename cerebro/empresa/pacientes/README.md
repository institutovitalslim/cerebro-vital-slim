# Pacientes — camada experimental / teste

Este diretório contém, neste momento, materiais de teste e validação operacional ligados à pré-consulta.

## Regra atual
Enquanto não forem promovidos a playbook, protocolo oficial ou base canônica, estes arquivos devem ser lidos como:
- material experimental
- teste de fluxo
- validação operacional

## Observação
Se algum formato aqui virar padrão oficial, ele deve ser promovido para a camada canônica ou operacional adequada.

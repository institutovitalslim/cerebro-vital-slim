# GitHub

## Repositório oficial do cérebro
- Owner/repo: `institutovitalslim/cerebro-vital-slim`
- Remote URL: `https://github.com/institutovitalslim/cerebro-vital-slim.git`

## Convenção
Quando Tiaro disser **"commit no cérebro"**, interpretar como:
1. atualizar os arquivos necessários no workspace `/root/cerebro-vital-slim`;
2. criar commit com mensagem clara;
3. publicar via `git push` no repositório oficial.

## Cuidados
- Confirmar o `origin` antes de fazer push se houver dúvida.
- Se o remoto estiver divergente, sincronizar com `origin/main` antes de publicar.
- Não assumir que outro repositório antigo ou genérico é o repo do cérebro.

# Critérios de Sucesso por Domínio

Definições objetivas de "concluído" para tarefas recorrentes.

## GitHub
Concluído quando, conforme o caso:
- issue/PR/comentário foi criado ou atualizado no repositório correto;
- branch, commit ou push existe de fato;
- CI/status foi consultado com evidência real;
- links, IDs ou números relevantes foram registrados na resposta.

## Quarkclinic
Concluído quando, conforme o caso:
- agenda correta foi consultada com dados reais;
- slot/horário foi confirmado com o endpoint correto;
- agendamento/consulta foi efetivamente criado, listado ou validado;
- quando houver ação externa derivada, os dados necessários ficaram prontos ou enviados com evidência.

## WhatsApp / Z-API
Concluído quando, conforme o caso:
- mensagem foi realmente enviada e existe evidência objetiva (`messageId`, log, status ou retorno real);
- se o contexto impedir envio externo, isso foi explicitado sem fingir execução;
- destinatários, conteúdo e canal ficaram claros;
- estado, log ou arquivo operacional foi atualizado quando aplicável.

## Omie
Concluído quando, conforme o caso:
- cadastro/emissão/faturamento foi criado no ambiente correto;
- IDs, banco, categoria e campos estruturados foram validados;
- não houve duplicidade indevida;
- boletos/PDFs e artefatos derivados foram baixados ou entregues quando isso fizer parte do fluxo.

## Tweet-carrossel
Concluído quando:
- memória e fontes obrigatórias foram consultadas;
- copy foi criada, refinada e aprovada quando necessário;
- capa e slides foram gerados no formato correto;
- arquivos finais existem nos caminhos esperados;
- entrega não foi apenas texto quando o pedido exigia imagens prontas.

## Configuração / gateway / automações
Concluído quando:
- config foi validada e aplicada sem quebrar o serviço;
- restart/reload ocorreu quando necessário;
- doctor/status/validação equivalente confirma o estado final;
- qualquer warning residual importante foi explicitado.

## Memória / cérebro
Concluído quando:
- aprendizado foi classificado;
- arquivo canônico correto foi atualizado;
- índice foi atualizado se necessário;
- ledger foi registrado;
- memória diária recebeu resumo quando fizer sentido.

## Regra geral
Nunca usar "concluído" apenas porque o comando rodou.
"Concluído" exige efeito real confirmado.

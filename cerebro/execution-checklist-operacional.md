# Checklist Operacional Rápida

Use esta checklist antes de responder ou executar tarefas operacionais críticas.

1. **Qual é o domínio?**
   - GitHub
   - Quarkclinic
   - WhatsApp/Z-API
   - Time da clínica

2. **Consultar a fonte canônica**
   - Ler `cerebro/OPERATIONS_INDEX.md`
   - Ler o arquivo específico do domínio
   - Ler `cerebro/verdades-operacionais.md` se houver qualquer dúvida

3. **Confirmar a ação real antes de prometer**
   - Se envolve envio, confirmar que o canal realmente entrega
   - Se envolve agenda, confirmar que a agenda correta está sendo usada
   - Se envolve GitHub, confirmar o repo/remoto corretos

4. **Só então responder ou executar**

## Regra de ouro
Se o dado é recorrente e importante, não confiar na memória isolada: consultar a fonte canônica primeiro.

# Checklist de Recuperabilidade Operacional

Antes de concluir que "não sabe fazer", verificar:
- isso já foi feito antes?
- existe skill relacionada?
- existe playbook relacionado?
- existe relatório anterior com mesmo tipo de entrega?
- existe termo alternativo que Tiaro usa para pedir isso?
- a tarefa já deveria estar na camada operacional?

## Regra
Se a resposta for sim para qualquer item acima, a prioridade é recuperar e executar — não responder com desconhecimento.

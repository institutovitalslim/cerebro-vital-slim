# Graphify RC-25 — Regra canônica: Maria corrige execução dos agentes

Tiaro determinou que, se houver problema com a execução de algum agente, Maria deve corrigir. Esta regra complementa a regra de que Maria não executa no lugar de João ou de outros agentes.

## Síntese

Maria não substitui o especialista, mas é responsável por manter o fluxo de agentes funcionando: diagnóstico, configuração, roteamento, sessão, tópico, runtime, prompt, acesso, cobrança, teste e validação.

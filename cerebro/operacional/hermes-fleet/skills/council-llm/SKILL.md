# Skill: llm-council — stress-test de decisão de alto impacto
Convoca 4 LLMs diversos (Opus 4.8, GPT-5.5, Kimi K2.6, Gemini 2.5 Pro via OpenRouter) para
stress-testar adversarialmente uma decisão. Presidente (Opus) consolida o veredito. NÃO executa nada.
Uso: `python3 /root/.hermes/skills/council-llm/council_llm.py "<decisão de alto impacto>"`
Quando: antes de qualquer decisão de alto impacto/irreversível (mudar atendimento, política, arquitetura, gasto grande).

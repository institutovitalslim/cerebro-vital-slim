# Skills — Vendas

| Skill | O que faz | Frequência |
|-------|-----------|-----------|
| `relatorio-vendas` | Lê planilha de vendas e gera relatório formatado | Diário (8h) via cron |
| `follow-up-leads` | Identifica leads sem contato há 3+ dias, sugere próximos passos | Diário (9h) via cron |


## Regra estrutural
- cada skill deve ser lida primeiro pelo `SKILL.md`
- scripts não substituem a interface da skill
- referências e assets são camadas auxiliares
- toda skill deve seguir `cerebro/empresa/skills/GOVERNANCA-SKILLS.md`

# Governança estrutural das áreas

Este diretório concentra regras comuns para leitura e organização de `cerebro/areas/`.

## Ordem correta de navegação
1. `MAPA.md`
2. `contexto/`
3. `rotinas/`
4. `skills/`
5. `projetos/`
6. experimental/teste ou histórico/log, quando explicitamente marcados

## Regra
Nenhuma área deve depender de arquivos soltos sem papel estrutural claro.

Unlike basic chatbots, these "claws" use a new runtime called OpenShell to operate within isolated sandboxes, ensuring that sensitive corporate data stays protected while the agents perform multi-step tasks like troubleshooting API failures or managing customer billing.

By pairing these agents with Nvidia’s new DGX Spark "desktop supercomputers," companies can run high-reasoning tasks locally to avoid the latency and high costs of cloud-based inference. This infrastructure shift signals Nvidia's intent to become the foundational layer for a "digital workforce" where autonomous agents don't just answer questions, but actively execute the administrative and technical labor of a global enterprise.
